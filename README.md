# FirePot (Backend)

This is a REST API Backend for use with the SPA Frontend (FirePot-Web).

Designed to be simple and easy to deploy via WSGI.


Libraries
----
* Flask
* SQLAlchemy
* JWT

Features:
----
* Authentication (JWT Token)
  * Login
  * Registstration
* Administration Pages
  * Inventory Management
* Store
  * List Inventory
  * Item Details
